== README

=== Requisites

1. To install mysql2 gem need: 
```
$ sudo apt-get install libmysqlclient-dev
```
2. Run:
```
$ bundle install
```

===  Setup MySQL DB

Run: 
```
$ rake mysql:create_schema
$ rake mysql:start
```
This tash create dev, test and prd schemas on local MySQL DB.

=== Run tests

```
$ rake test
```
